# Project-Game-Pingpong-By-Benneth
This is my first time trying to make some games , hope you guys have fun playing this game
Description of the game: 
1. Genre : simulator
2. Age : Every Age
3. Release : 19 July 2026
4. Every thing inside the game : 2 rackets and a pingpong ball 

By the way , this are the steps you need to know in order to play this game with the best experience: 
1. inside this game , you will find 2 rackets or pingpong paddles , this is used to bounce the ball back
2. The ball always moves constantly , so you better watch out
3. If the ball goes through you defenses , you will lose

Conclusion : This game is a pingpong game where you have to defend your place from the ball , if you missed the ball , you lose
